# Doom Mystery Game
A C++ college project inspired by social deduction games like Among Us.

## Description
Doom Mystery Game is a onsole-based game where players complete tasks while trying to identify the imposter.

## Features
Multiplayer-like logic simulation
Task completion system
Mystery-solving gameplay

## Tech Stack
Languages C++
Tools VS Code

## How to run
1. Open the project folder in a C++ IDE (e.g, VS Code)
2. Compile 'main.cpp'
3. Run the program and follow console instructions
